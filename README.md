## Antes de subir
1. Instala dependencias con `npm install`
2. Crea un archivo `.env` usando `.env.example`
3. Coloca tu `TELEGRAM_BOT_TOKEN` y `TELEGRAM_CHAT_ID`
4. Ejecuta `npm start`

## Hosting
Necesitas un hosting que soporte Node.js porque el envio a Telegram ya no sale directo desde el navegador.
